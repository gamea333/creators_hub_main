# Admin Approvals & Moderation Panel

A modern and responsive **Admin Dashboard** built with **React + Vite** to streamline the moderation workflow for internal teams. The application enables administrators to efficiently review pending approvals, moderate content, perform bulk actions, and maintain a clear activity history through an intuitive interface.

## 📌 Project Overview

This project was developed as part of a frontend assignment focused on designing and implementing an efficient moderation dashboard.

The interface prioritizes speed, clarity, and usability, allowing administrators to quickly review creators, campaigns, and content while minimizing friction during the approval process.

---

## ✨ Features

* Pending approvals dashboard
* Detailed review page for each submission
* Approve, Reject, and Request Changes actions
* Search functionality
* Filter submissions by status and category
* Activity log for moderation history
* Bulk selection and bulk actions
* Responsive design
* Clean and modern UI
* Fast loading with Vite

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/admin-approvals-moderation.git
```

Navigate to the project

```bash
cd admin-approvals-moderation
```

Install dependencies

```bash
npm install
```

or

```bash
bun install
```

Start the development server

```bash
npm run dev
```

or

```bash
bun run dev
```

Build for production

```bash
npm run build
```

or

```bash
bun run build
```

Preview production build

```bash
npm run preview
```
